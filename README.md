📌 Task Manager App (MERN Stack)

A full-stack Task Manager application built with MongoDB, Express, React, and Node.js (MERN).
It allows users to create, update, delete, and manage tasks with authentication support.

🚀 Features
🔐 User Signup & Login
📝 Create new tasks
✏️ Edit & update tasks
🗑️ Delete tasks
☑️ Mark tasks as completed
🎯 Filter tasks (All / Active / Completed)
⚡ Optimistic UI updates
🎨 Modern responsive UI with Tailwind CSS
🔄 REST API integration with backend
🛠️ Tech Stack
Frontend
React.js
Tailwind CSS
Axios
React Router DOM
Backend
Node.js
Express.js
MongoDB
Mongoose
CORS
📁 Project Structure
Task-manager-app/
│
├── client/        # React Frontend
│   └── src/
│       ├── pages/
│       ├── api/
│
├── Server/        # Node + Express Backend
│   ├── models/
│   ├── routes/
│   ├── server.js
│
└── README.md
⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/your-username/task-manager-app.git
cd task-manager-app
2️⃣ Setup Backend
cd Server
npm install

Create .env file (optional):

MONGO_URI=mongodb://127.0.0.1:27017/task-manager
PORT=5000

Run backend:

node server.js
# or
nodemon server.js
3️⃣ Setup Frontend
cd client
npm install
npm start
🌐 API Endpoints
Auth Routes
Method	Endpoint	Description
POST	/api/auth/signup	Register user
POST	/api/auth/login	Login user
Task Routes
Method	Endpoint	Description
GET	/api/tasks	Get all tasks
POST	/api/tasks	Create task
PUT	/api/tasks/:id	Update task
DELETE	/api/tasks/:id	Delete task
